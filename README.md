The Online Code Editor is a web-based application built using HTML, CSS, and JavaScript. It allows users to write HTML, CSS, and JavaScript code in separate editor sections and execute it using a Run button. The combined code is rendered instantly in a live preview panel using an iframe.
The project features a dark-themed interface, line numbers for better readability, and a simple, user-friendly layout. It demonstrates core frontend development concepts such as DOM manipulation, event handling, and dynamic content rendering
